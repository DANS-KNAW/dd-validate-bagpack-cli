dd-validate-bagpack-cli
========================

Provides the `bagpack-validate` command to interact with `dd-validate-bagpack` service via its [REST API]{:target=_blank}.

SYNOPSIS
--------

```bash
bagpack-validate [ -n | --no-wait ] [ -i | --poll-interval <ms> ] <bag-path>
```

For more information use:

```bash
bagpack-validate --help
```

[REST API]: {{ validate_bagpack_api }}
